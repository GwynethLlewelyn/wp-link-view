=== Link View - Wordpress Plugin ===
Contributors: mibuthu
Tags: link, links, list, images, pictures, integrated, page, categories, banner, slideshow, sponsor, widget
Requires at least: 3.0
Tested up to: 3.3.1
Stable tag: trunk

The purpose of this plugin is to to show the wordpress integrated links in a list or in a slideshow. It is also possible to include the images. Use the shortcode [linkview] to add the Links/Blogroll in your site. There is also a widget available.
